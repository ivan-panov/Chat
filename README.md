# Chat Widget

**Современный WordPress-плагин онлайн-чата с оператором, Telegram, MAX и CRM**

Лёгкий, быстрый и функциональный плагин чата для WordPress с мощными интеграциями.

## ✨ Основные возможности

### Для посетителей сайта
- Красивый floating-чат (ленивая загрузка скриптов)
- Отправка текста, изображений и файлов (до 20 МБ)
- Звуковые уведомления
- Автоматическое извлечение email и телефона из сообщений
- Мобильная адаптация

### Для оператора (панель в админке)
- Реал-тайм обновление диалогов и сообщений
- Просмотр геолокации (страна, город, регион, провайдер, IP)
- Автоматическое извлечение контактов
- Статусы прочтения (✓ и ✓✓)
- Разделители по датам
- Закрытие и полное удаление диалогов
- Лёгкий просмотр изображений в модальном окне

### Интеграции

#### 📱 Telegram
- Уведомления о новых сообщениях
- Ответы прямо из Telegram
- Команды и интерактивные кнопки
- Обработка файлов и изображений
- Надёжный webhook + фоновый worker (очередь)
- Поддержка SOCKS5-прокси
- Защита через секретный токен

#### 💬 MAX (VK)
- Уведомления с интерактивными кнопками:
  - ✉ **Ответить**
  - 📜 **История диалога**
  - ❌ **Закрыть**
  - 📊 **Статистика**
  - 💳 **СБП QR**
- Автопривязка оператора через команду `/start`

#### 🔗 CRM API (двусторонний)
- Отправка событий: `dialog.created`, `message.created`
- Входящий REST API для внешних CRM-систем
- Возможность получать список диалогов, отправлять сообщения и закрывать диалоги из CRM
- HMAC SHA-256 подпись + Bearer-токен авторизация

## 🚀 Установка

1. Скачайте последний релиз
2. WordPress → **Плагины** → **Добавить новый** → **Загрузить плагин**
3. Активируйте плагин
4. Перейдите в меню **Чат** в админ-панели

## ⚙️ Настройка

### Telegram
1. Создайте бота через [@BotFather](https://t.me/botfather)
2. Укажите **Bot Token** и **Admin Chat ID**
3. Нажмите **Подключить Webhook**
4. (Опционально) настройте SOCKS5-прокси

### MAX
1. Получите токен бота в MAX
2. Укажите токен в настройках
3. Напишите боту команду `/start` — User ID автоматически привяжется

### CRM
- Укажите **Endpoint URL** вашей CRM
- Настройте токен авторизации и секрет подписи

## 📋 Требования

- WordPress 5.8+
- PHP 7.4+
- Расширение cURL (для прокси и API-запросов)
- MySQL / MariaDB

## 🛠 Технологии

- Чистый PHP + WordPress REST API
- Реал-тайм обновления через polling + AJAX
- Фоновый обработчик очереди (Telegram)
- Защищённые webhook'и с секретами
- Ленивая загрузка фронтенд-скриптов
